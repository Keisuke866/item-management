# 商品管理システム

## 概要
このシステムでは、店舗で扱う商品の在庫管理を行うことが出来ます。

## 主な機能
- ログイン・ログアウト機能
- 商品一覧機能
- 商品新規登録、売却した商品と在庫商品の選別、削除機能
- 商品検索機能
- 商品在庫管理機能

### 開発環境
- PHP 8
- Mysql 14.14
- Laraevl 8.75
